# f-cloud

### 介绍
简单、高效 spring cloud 快速开发前后端分离项目

### 使用技术
- spring boot 2.6
- spring cloud alibaba
- spring cloud Gateway
- nacos 
- sentinel
- jwt
- mybatis-plus
- redis
- jetcache
- validation
- mapstruct
- disruptor
- minIO

### 软件架构
- api 接口、实体定义
- common 公共模块（工具类等）
- service 服务实现包含dao
- gateway 网关

### 使用端口
- gateway 8000
- sys 18080
- message 18081
- file 18082
- sell 18083
- order 18084
- pay 18085
